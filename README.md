# 準備

## MySQL を用意する

### インストール

```
brew install mysql
```

### セットアップ

```
mysql.server start
mysql_secure_installation
```

## Sequel Pro を用意する

- MySQL の中身を見るための GUI ツール
- 無料。Mac で使える。
- https://www.sequelpro.com/
